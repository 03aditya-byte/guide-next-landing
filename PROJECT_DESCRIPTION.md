# Career Compass - Project Description

## Overview
Career Compass is a modern, responsive web application designed to provide comprehensive career guidance and counseling services. The platform serves as a bridge between students seeking career direction and professional counselors offering expert guidance.

## Project Purpose
The application addresses the growing need for personalized career guidance in today's rapidly evolving job market. It provides students with AI-powered career recommendations while connecting them with professional counselors for personalized support.

## Key Features

### Landing Page
- **Hero Section**: Compelling introduction with clear value proposition
- **Statistics Dashboard**: Showcases platform success metrics (10,000+ students guided, 98% success rate)
- **Feature Highlights**: Four core benefits presented with engaging icons and descriptions
- **Call-to-Action Sections**: Strategic placement of conversion elements

### User Panel (Student Portal)
- **Personalized Dashboard**: 
  - Displays recommended career paths as animated cards
  - Progress tracking for career exploration journey
  - Quick access to key features
- **Career Explorer**:
  - Skill-based filtering system
  - Dynamic career recommendations
  - Interactive career path cards with detailed information
- **Counseling Services**:
  - Grid layout of available counselors
  - Profile cards with expertise areas
  - Mock calendar integration for session scheduling

### Admin Panel
- **Analytics Dashboard**:
  - Key performance metrics (total users, sessions booked, active counselors)
  - Clean, data-focused interface
- **Resource Management**:
  - Add, view, and delete career resources
  - Comprehensive resource library management
- **Counselor Management**:
  - Full CRUD operations for counselor profiles
  - Expertise and contact information management

## Technical Specifications

### Technology Stack
- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite for fast development and optimized builds
- **Styling**: Tailwind CSS with custom design system
- **UI Components**: shadcn/ui component library
- **Routing**: React Router DOM for navigation
- **State Management**: React Query for server state
- **Icons**: Lucide React for consistent iconography

### Design System
- **Color Palette**: Professional blue, gray, and white theme
- **Typography**: Hierarchical text system with clear readability
- **Animation**: Smooth transitions and hover effects throughout
- **Responsive Design**: Mobile-first approach ensuring cross-device compatibility

### Key Design Principles
- **Minimalist Aesthetic**: Clean, professional interface reducing cognitive load
- **Accessibility**: High contrast ratios and semantic HTML structure
- **Performance**: Optimized animations and efficient rendering
- **User Experience**: Intuitive navigation with clear user journeys

## User Experience Flow

### Student Journey
1. **Discovery**: Land on homepage and explore platform benefits
2. **Registration**: Access student portal (simulated)
3. **Exploration**: Use Career Explorer to discover paths based on skills
4. **Guidance**: Connect with counselors for personalized advice
5. **Progress**: Track career development through dashboard

### Admin Workflow
1. **Dashboard Overview**: Monitor platform metrics and activity
2. **Resource Curation**: Manage career guidance materials
3. **Counselor Network**: Maintain counselor database and profiles
4. **Platform Management**: Oversee overall platform health
